# 소개

이 블로그는 수학, 과학, 프로그래밍에 관한 기록을 남기기 위해 만들었다.
NieR:Automata의 UI를 참조한 테마 위에서, 수식과 코드가 많은 글을 편하게
읽을 수 있도록 구성했다.

# 블로그 구성

- **홈**: 전체 게시물 목록
- **태그**: 주제별 분류
- **에디터**: 로컬 환경에서 글을 작성·편집·배포하는 전용 페이지

> 이 블로그는 React + Vite로 만들어졌으며, GitHub Pages에서 호스팅된다.
> 차후 Three.js 기반의 3D UI로 확장할 예정이다.
